# Pet Care & Products Web Application

## Overview
This project is a full-stack web application designed for pet owners, combining pet-related services, information, and online shopping into one centralized platform. The application allows users to create accounts, browse different categories, add products to a shopping cart, and complete purchases through an integrated checkout and payment system.

The project was developed using **Django** for the backend together with **HTML**, **CSS**, and **JavaScript** for the frontend.

---

## Features

### User Authentication
- User registration and profile creation
- Login and logout functionality
- Personalized user experience

### Navigation System
The application contains a sidebar navigation menu with categories such as:
- Veterinarians
- Pet Care
- Pet Food
- Products
- Contact

Each category redirects users to a dedicated page containing useful information, explanations, and related functionalities.
